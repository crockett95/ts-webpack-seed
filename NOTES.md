## Commands

1. `watch`
  * Autowatch file compilation
  * Auto run tests
    * Unit tests on file change
    * Integration tests on start
  * Lint on file change
2. `serve`
  * `gulp watch` plus test server
3. `build`
  * Lint
  * Compile all TS files minified
    * strip console statements
  * Compile all SCSS files minified
  * Write `.d.ts` files
  * Build docs
    * typedoc
    * sassdoc
4. `test`
  * Lint files
  * Run all tests once
    * Unit
    * Integration
    * Layout?
